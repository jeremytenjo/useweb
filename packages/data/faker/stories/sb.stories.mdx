# faker

```js
import faker from '@useweb/lib/data/faker'
```

### Example

```js
import faker from '@useweb/lib/data/faker'

const stub = () => ({
  name: faker.name.findName(),
  email: faker.internet.email(),
  teamId: faker.datatype.uuid(),
  leagueId: faker.datatype.uuid(),
  position: 'ST',
  avatar: faker.image.image(),
})
```
